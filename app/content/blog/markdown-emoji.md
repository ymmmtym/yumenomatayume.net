Markdown で絵文字を使用する際に便利なツールやショートカットをまとめました 😊

## 絵文字チートシート

絵文字のコードを調べる際は、以下のサイトが便利です：

- [🎁 Emoji cheat sheet for GitHub, Basecamp, Slack & more](https://www.webfx.com/tools/emoji-cheat-sheet/)

GitHub や Slack などで使用できる絵文字コード（`:smile:` など）を簡単に検索できます 🔍

## Markdown エディター

**Typora** というエディターを使用すると、Markdown のプレビューを確認しながら編集できます ✨

絵文字がどのように表示されるかをリアルタイムで確認できるため、とても便利です。

## Mac での絵文字入力ショートカット

Markdown ファイルに限らず、Mac では以下のショートカットキーで絵文字パレットを開けます：

**`Command + Control + Space`** 🎯

このショートカットを覚えておくと、どのアプリケーションでも素早く絵文字を挿入できます。

## 使用例

```markdown
# プロジェクトの進捗 📊

- ✅ 設計完了
- 🚧 開発中
- ⏳ テスト待ち
- 🎉 リリース完了
```

絵文字を適切に使用することで、文書がより読みやすく、親しみやすくなります 💡
