GitHub の可愛いマスコット「Octocat」をオリジナルでカスタマイズできることをご存知でしょうか？ 🐙 今回は、自分だけの Octocat を作成する方法をご紹介します。

## Octocat とは

GitHub のオフィシャルマスコットになっているタコみたいな猫みたいなキャラクターの**種族**です 🐱 正式名称は**「Monalisa」**と呼ばれています。

## オリジナル Octocat を作成する

[公式サイト](https://myoctocat.com/)にアクセスして「GET STARTED」ボタンを押すと、すぐに作成を開始できます ✨

作成画面では、以下の要素をカスタマイズできます：
- 体の色
- 目の形
- アクセサリー
- 背景

## 作成した Octocat を保存する

完成したら「SHARE」アイコンから、Twitter やメールで共有することができます 📤

画像として保存したい場合は、メール送信を試してみましたが、なかなか届かない場合があります。そのため、作成した Octocat はスクリーンショットで保存して使用するのがおすすめです 📸

オリジナルの Octocat で、あなたの GitHub プロフィールをより個性的に演出してみてください！ 🚀
